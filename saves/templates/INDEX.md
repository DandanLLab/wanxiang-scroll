- **文件**：[character.md](./character.md) | [character.json](./character.json)

## 模板使用方法

### 1. 变量替换
所有模板使用 `${variable}` 作为占位符，使用时替换为实际值：

```markdown
- **世界名称**：${world_name}
```

替换后：
```markdown
- **世界名称**：艾尔德兰大陆
```

### 2. JSON 格式
JSON 模板用于程序处理，保持数据结构：

```json
{
  "world_name": "${world_name}",
  "world_type": "${world_type}"
}
```

### 3. 程序调用
```python
from string import Template

template = Template(open('world_gen.md').read())
result = template.substitute(
    world_name="艾尔德兰大陆",
    world_type="奇幻中世纪",
    era="第二纪元",
    start_year=1250
)
```

## 模板扩展

### 创建新模板
1. 复制现有模板作为基础
2. 修改字段和结构
3. 保持 `${variable}` 占位符格式
4. 同时创建 .md 和 .json 版本

### 模板命名规范
- 使用小写字母和下划线
- 描述性名称，如 `world_gen`、`character`
- .md 用于人类阅读，.json 用于程序处理
