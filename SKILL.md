# 万象绘卷

全能故事创作工具箱，支持网文创作、交互式故事、56种文风切换、剧情管理、拆书融合创作。

## 核心功能

- **56种文风切换**：殿堂文学、流行通俗、二次元、科幻、恐怖等12大区
- **网文创作全流程**：黄金三章、细纲生成、平台适配（起点/番茄/知乎/晋江）
- **交互式故事模拟**：实时互动、角色扮演、世界探索
- **剧情管理**：MoM系统、大总结、多故事线管理
- **拆书融合创作**：六维要素提取、七步循环法、四换三原则
- **质量控制**：去AI化处理、评审标准、编辑技巧

## 快速开始

### 创作模式
```
用户：帮我用武侠文风写一段小说开头
系统：调用文风系统，生成武侠风开头
```

### 交互模式
```
输入：打开菜单
系统：启动交互式故事界面
```
