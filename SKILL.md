# 万象绘卷

> 全能故事创作与交互体验工具箱

## 功能概览

- **56种文风切换**：从殿堂文学到网文爽文，一键切换
- **网文创作全流程**：黄金三章、细纲生成、平台适配
- **交互式故事模拟**：实时互动、角色扮演
- **剧情管理系统**：长篇故事管理、大总结系统
- **拆书融合创作**：多源小说融合创作新作品
- **小说档案馆**：爬取、分析、存储小说素材

## ⚠️ 透明声明

### 爬虫脚本说明

本技能包含以下爬虫脚本（位于 `scripts/` 目录）：

| 脚本 | 目标站点 | 用途 |
|------|----------|------|
| `crawl_novel_index.py` | novel.transchinese.org | 爬取分类索引与下载txt |
| `crawl_jh_novel.py` | 刺猬猫、起点等多平台 | 聚合小说搜索下载 |
| `crawl_fq_novel.py` | fanqienovel.com | 番茄小说搜索下载 |

**重要声明**：
- 所有爬虫脚本**仅供学习参考使用**
- 用户需自行确认目标站点的许可与版权合规
- 爬取内容不得用于商业用途
- 建议限制爬虫范围与频率，遵守 robots.txt

### 外部 API 声明

- 本技能**不包含任何外部 API 调用**
- 所有配置文件仅用于本地处理
- **无外部端点、无第三方服务**

### 凭据要求

- **无需任何环境变量或凭据**
- **无需 API 密钥**
- 所有功能均为本地运行

### 安全约束
