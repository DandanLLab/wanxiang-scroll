# 拆书融合创作

> 本章为用户自主拆书分析空间，所有分析报告由用户自行创建和维护。

---

## 📖 章节说明

本章节用于存放用户自行拆书分析的小说报告。你可以：

1. **分析喜欢的小说**：提取核心设定、人物、剧情等要素
2. **学习优秀作品**：研究作者的写作技巧和叙事方法
3. **积累创作素材**：为融合创作储备灵感和参考

---

## 🚀 快速开始

### 如何拆书？

1. 选择一本你想分析的小说
2. 阅读并提取关键要素：
   - 核心设定（世界观、力量体系等）
   - 人物设定（主角、配角、关系网）
   - 剧情结构（主线、支线、高潮点）
   - 写作技巧（对话风格、叙事节奏）
3. 在 `reports/` 目录下创建分析报告
4. 整理可借鉴的创新点和融合建议

### 报告模板

每份分析报告可包含以下内容：

```
一、基本信息
   - 小说名称、作者、字数、类型

二、核心设定
   - 世界观、力量体系、特殊设定

三、人物设定
   - 主角、配角、人物关系

四、核心看点
   - 吸引读者的关键要素

五、可借鉴的创新点
   - 独特的设定和创意

六、素材价值评估
   - 可用于融合创作的素材

七、融合建议
   - 如何与其他作品融合
```

---

## 💡 使用建议

1. **选择性分析**：不必分析所有小说，选择对你有帮助的即可
2. **深度优先**：深入分析几本优秀作品比浅尝辄止更有价值
3. **持续更新**：随着创作需要，不断补充新的分析报告
4. **分类整理**：可以按类型、风格等维度组织报告
