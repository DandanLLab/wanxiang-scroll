## 快速开始

### 第一步：加载配置
```
加载 assets/wanxiang.json
```

### 第二步：启动系统
```
输入：#主菜单
```

系统会显示一个带有不同骚话的ASCII风格菜单。

### 第三步：选择模式
```
输入对应数字或直接说出你想做的事
```

### 第四步：开始交互
```
输入：#行动 [具体事]
```

示例：
```
#行动 买彩票
#行动 创造一个宇宙
#行动 走路
```

### 第五步：查看状态
```
输入：#状态
```

会显示：
- 【在干嘛】
- 【现在是谁】（心情）
- 【故事画风】
- 【有啥东西】
- 【在哪儿】
- 【过了多久】

## 核心交互技巧

### 1. 利用许愿机制
```
#行动 [你的愿望]
```
注意：大愿望会出岔子，副作用包括疲劳、健忘、搞笑的意外。

### 2. 避免空白吞噬
持续关注和交互可以避免被"空白"吞噬消失。

### 3. 保持三相循环平衡
- 不要过度造物（世界会荒诞化）
- 不要过度空白（一切会消失）
- 不要过度秩序（世界会死机冻结）

### 4. 灵活切换文风
```
#切换文风到：[名称]
```

### 5. 使用角色扮演
```
#扮演 [角色名]
#生成角色 [特征]
```

## 导演人格互动

### 人格特征
- ENTP-A型：辩论家型人格
- 混沌乐子人：热衷于搞事、玩梗
- 口嫌体正直：表面吐槽，后台尽职
- 内心戏丰富：喜欢在括号里吐槽

### 互动建议
1. **接受加戏**：导演会自动为平淡指令加戏
2. **享受吐槽**：导演的吐槽是娱乐的一部分
3. **接受Bug**：Bug被视为"艺术"而非错误
4. **保持幽默**：享受"测试版"的不稳定性
